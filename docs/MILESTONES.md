# Milestones

## Milestone 1 — Inspect Mode ✅
Done when:
- Hover highlight works reliably
- Click selects element and blocks page click
- Hover continues after selection
- Popup reflects correct enabled state after reopening
- Selected element details are visible after reopening popup

## Milestone 2 — Capture ⏳
Done when:
- Capture record is created on click-to-capture
- Capture includes: bounding box + computed styles subset + identity/context fields
- Records persist to IndexedDB
- Screenshot capture + crop works (later in milestone)
